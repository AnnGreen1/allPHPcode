在以上的配置中，我们需要修改其中的 setup.php 文件，文件的内容如下：

xxxxxxxxxx
<?php
/*
 * @Description: Smarty 配置类
 * @Version: 1.0
 * @Author: Humbert Cheung
 * @Date: 2021-11-10 09:10:09
 * @LastEditors: [Humbert Cheung]
 * @LastEditTime: 2021-11-14 17:10:21
 * @FilePath: /management-system(Smarty)/helpers/setup.php
 * Copyright (C) 2021 syzhang. All rights reserved.
 */


define('BASE_DIR', $_SERVER['DOCUMENT_ROOT']); // 获取服务器站点目录
define('APP_DIR', BASE_DIR . '/management-system(Smarty)/'); // 获取程序根目录


require_once(APP_DIR . 'third_party/Smarty/Smarty.class.php');


class MS_Smarty extends Smarty
{
  function MS_Smarty()
  {
    //类构造函数.创建实例的时候自动配置
    $this->__construct();


    $this->template_dir = APP_DIR . 'app/view';
    $this->compile_dir = APP_DIR . 'cache/smarty/compiled';
    $this->config_dir = APP_DIR . 'config';
    $this->cache_dir = APP_DIR . 'cache/smarty/cached';


    $this->assign('app_name', 'Smarty Extend');
  }
  /**
   * 启用缓存
   *
   * Allows you to enable caching on a page by page basis
   */
  public function enable_caching()
  {
    $this->caching = true;
  }


  /**
   * 禁用缓存
   *
   * Allows you to disable caching on a page by page basis
   */
  public function disable_caching()
  {
    $this->caching = false;
  }
}

 

针对其中缓存的启用与禁用，通过封装两个方法来实现，后续在使用过程中，只需要调用对应的方法即可。

1、登录模块重构

1.1、登录控制器设计

用户从入口文件发送指令后，需要将用户重定向到 login.php 页面，那么在该控制器（LoginController.php）中，我们首先先将原来的 index() 方法进行修改，然后添加登录验证方法以及退出登录方法，如下：

 

xxxxxxxxxx
<?php
/*
 * @Description: 登录控制器
 * @Version: 1.0
 * @Author: Humbert Cheung
 * @Date: 2021-11-14 15:40:30
 * @LastEditors: [Humbert Cheung]
 * @LastEditTime: 2021-11-15 21:08:44
 * @FilePath: /management-system(Smarty01)/app/controller/LoginController.php
 * Copyright (C) 2021 syzhang. All rights reserved.
 */
require_once './app/model/LoginModel.php';


class LoginController
{
  /**
   * @description: 登录页面引导方法
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function index()
  {
    global $smarty;
    $smarty->assign('login_action', './index.php?controller=Login&method=login');
    $smarty->display('login.html');
  }


  /**
   * @description: 登录验证方法
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function login()
  {
    $username = $_REQUEST['username'];
    $password = $_REQUEST['password'];


    $loginModel = new LoginModel();
    $user = $loginModel->getUser($username);
    // 判断数据是否为空
    if (empty($user)) {
      // 弹窗提示用户不存在，返回到登录页面
      echo "<script>alert('用户不存在！');history.go(-1);</script>";
    } else {
      // 用户存在
      if ($user['password'] == $password) {
        // 密码正确，跳转到首页
        // 启用session
        session_start();
        // 将用户名保存到session中
        $_SESSION['username'] = $username;
        echo "<script>location.href='index.php?controller=Users&method=userslist';</script>";
      } else {
        // 弹窗提示密码错误，返回到登录页面
        echo "<script>alert('密码错误！');history.go(-1);</script>";
      }
    }
  }
  /**
   * @description: 退出登录方法
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function logout()
  {
    session_start();
    unset($_SESSION['username']);
    echo "<script>alert('退出成功！');location.href='index.php';</script>";
  }
}

1.3、登录模型设计

在登录模型中，我们主要通过模型来帮我们获取登录的数据，因此登录模型主要是负责到数据库中验证用户信息，并将结果返回，LoginModel.php 如下：

 

xxxxxxxxxx
<?php
/*
 * @Description: 登录模型类
 * @Version: 1.0
 * @Author: Humbert Cheung
 * @Date: 2021-11-14 20:21:40
 * @LastEditors: [Humbert Cheung]
 * @LastEditTime: 2021-11-14 20:48:40
 * @FilePath: /management-system(Smarty)/app/model/LoginModel.php
 * Copyright (C) 2021 syzhang. All rights reserved.
 */
class LoginModel
{
  /**
   * @description: 根据用户名获取用户
   * @param {*} $username 用户名
   * @return {*} $user 用户
   * @Author: Humbert Cheung
   */
  public function getUser($username)
  {
    // 获取全局 pdo 对象
    global $pdo;
    // 准备sql语句
    $sqlStr = "SELECT * FROM admin WHERE username = '$username'";
    // 执行sql查询
    $stmt = $pdo->query($sqlStr);
    // 提取结果集中数据
    $user = $stmt->fetch();
    // 返回数据
    return $user;
  }
}

 

至此，我们完成了登录模块的重构。

2、用户列表模块重构

2.1、用户列表控制器设计

当用户进入用户模块时，首先展示的是用户列表，因此，在该控制器（UsersController.php）中，我们首先需要实现 userslist( ) 方法，通过此方法来获取数据。不过在获取数据之前我们需要进行登录拦截，因此再添加一个 isLogin( ) 方法来实现登录拦截，同时也添加用户增删改查对应的方法。如下：

xxxxxxxxxx
<?php
/*
 * @Description: 用户列表控制器
 * @Version: 1.0
 * @Author: Humbert Cheung
 * @Date: 2021-11-14 20:51:47
 * @LastEditors: [Humbert Cheung]
 * @LastEditTime: 2021-11-15 21:25:54
 * @FilePath: /management-system(Smarty01)/app/controller/UsersController.php
 * Copyright (C) 2021 syzhang. All rights reserved.
 */
require_once './app/model/UsersModel.php';


class UsersController
{
  /**
   * @description: 登录拦截
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function isLogin()
  {
    session_start();
    if (empty($_SESSION['username'])) {
      // 找不到session中的用户名，说明用户是没有登陆过，那么就需要弹窗提醒，并跳转到登录页面
      echo "<script>alert('请先登录！');location.href='index.php'</script>";
    }
  }
  /**
   * @description: 用户列表
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function userslist()
  {
    $this->isLogin();
    global $smarty;
    // 实例化模型
    // 获取列表数据
    $usersModel = new UsersModel();
    // 获取列表数据
    $users = $usersModel->getUsers();
    // 存储到数组中
    $smarty->assign('data', $users);
    // 展示页面
    $smarty->display('userslist.html');
  }
  /**
   * @description: 跳转至添加用户页面
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function add()
  {
    global $smarty;
    $smarty->display('useradd.html');
  }


  /**
   * @description: 添加用户处理方法
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function addUser()
  {
    // 获取用户列表模型
    $usersModel = new UsersModel();
    // 添加新用户
    $count = $usersModel->addUser();
    // 判断执行的结果
    if ($count > 0) {
      echo "<script>alert('添加成功！');location.href='./index.php?controller=Users&method=userslist'</script>";
    } else {
      echo "<script>alert('添加失败！');location.href='./index.php?controller=Users&method=add'</script>";
    }
  }


  /**
   * @description: 跳转到编辑用户页面
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function edit()
  {
    // 获取用户模型
    $usersModel = new UsersModel();
    // 获取被编辑的用户信息
    $user = $usersModel->getEditingUser();
    global $smarty;
    $smarty->assign("user", $user);
    $smarty->display('useredit.html');
  }


  /**
   * @description: 编辑用户处理方法
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function editUser()
  {
    // 获取用户列表模型
    $usersModel = new UsersModel();
    // 编辑用户
    $count = $usersModel->editUser();
    // 判断执行的结果
    if ($count > 0) {
      echo "<script>alert('修改成功！');location.href='./index.php?controller=Users&method=userslist'</script>";
    } else {
      echo "<script>alert('修改失败！');location.href='./index.php?controller=Users&method=edit'</script>";
    }
  }


  /**
   * @description: 删除用户处理方法
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function delete()
  {
    // 获取用户列表模型
    $usersModel = new UsersModel();
    // 删除用户
    $count = $usersModel->deleteUser();
    $msg = "删除失败！";
    if ($count > 0) {
      $msg = "删除成功！";
    }
    echo "<script>alert('$msg');location.href='./index.php?controller=Users&method=userslist'</script>";
  }
}

class UsersModel
{
  /**
   * @description: 获取用户列表
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function getUsers()
  {
    global $pdo;
    // 准备sql语句
    $sql = "SELECT * FROM user";


    // 执行sql查询
    $stmt = $pdo->query($sql);
    $users = $stmt->fetchAll();
    $data = [];
    foreach ($users as $u) {
      // 对类型进行转化
      switch ($u['type']) {
        case 1:
          $u['type_name'] = "普通员工";
          break;
        case 2:
          $u['type_name'] = "项目经理";
          break;
        case 3:
          $u['type_name'] = "项目组长";
          break;
        default:
          break;
      }
      $u['icon'] = $u['icon'] ? $u['icon'] : 'default.jpeg';
      $u['ctime'] = date('Y-m-d H:i', $u['create_time']);
      array_push($data, $u);
    }
    return $data;
  }
  /**
   * @description: 添加用户
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function addUser()
  {
    global $pdo;
    $name = $_REQUEST['name'];
    $age = $_REQUEST['age'];
    $birthday = $_REQUEST['birthday'];
    $type = $_REQUEST['type'];
    $icon = $_REQUEST['icon'];
    $create_time = time();


    $sql = "INSERT INTO user(name, age, birthday, type, icon, create_time) VALUES('$name', $age, '$birthday', $type, '$icon', $create_time)";
    // 执行插入，获取结果
    $count = $pdo->exec($sql);
    return $count;
  }
  /**
   * @description: 获取需要被编辑的用户
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function getEditingUser()
  {
    global $pdo;
    // 获取用户信息
    $id = $_REQUEST["id"];
    $sql = "SELECT * FROM user WHERE id='$id'";
    $stmt = $pdo->query($sql);
    $user = $stmt->fetch();
    return $user;
  }


  /**
   * @description: 编辑用户
   * @param {*}
   * @return {*}
   * @Author: Humbert Cheung
   */
  public function editUser()
  {
    global $pdo;
    $id = $_REQUEST['id'];
    $name = $_REQUEST['name'];
    $age = $_REQUEST['age'];
    $birthday = $_REQUEST['birthday'];
    $type = $_REQUEST['type'];
    $icon = $_REQUEST['icon'];
    $create_time = time();


    $sql = "UPDATE user SET name='$name', age=$age, birthday='$birthday', type=$type, icon='$icon', create_time=$create_time WHERE id='$id'";


    $count = $pdo->exec($sql);
