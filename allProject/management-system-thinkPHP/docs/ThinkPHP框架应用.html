                                                                      4.1、 创建 Admin 模型

                                                                      以上虽然完成了登录页面的展示，不过对于登录验证等业务逻辑的实现，需要在 Login 控制器中定义对应的 login 操作来帮我们实现。那么我们需要在登录操作中，根据请求过来的用户名和密码，然后到数据库中完成相应的查询操作，最后进行验证和判断。不过对于数据库的操作，我们需要通过对应的模型类来帮我们实现，因此我们需要先定义管理员模型类 Admin 类，并且因为管理员类将来可能还会在其他模块中用到，因此我们可以在应用目录中添加 common 目录，然后在 common 中定义 model 目录，最后把 Admin 类定义 model 中：

                                                                      

                                                                      namespace app\common\model;
                                                                      

                                                                      use think\Model;
                                                                      

                                                                      class Admin extends Model
                                                                      {
                                                                      }

                                                                      那么，在后续的所有涉及到 admin 表的操作，都交由 Admin 模型来完成。

                                                                      4.3、实现 login 操作

                                                                      接着，我们就在控制器中定义 login 操作：

                                                                      namespace app\admin\controller;
                                                                      

                                                                      use think\Controller;
                                                                      use app\common\model\Admin as AdminModel;
                                                                      use think\facade\Session;
                                                                      

                                                                      class Login extends Controller
                                                                      {
                                                                        public function index()
                                                                        {
                                                                          // 自动定位渲染视图： 当前模块/view/当前控制器名/当前操作.html
                                                                          return $this->fetch();
                                                                        }
                                                                        public function login($username, $password)
                                                                        {
                                                                          // 通过模型查询用户数据
                                                                          $user = AdminModel::where('username', $username)->find();
                                                                          if(empty($user)) {
                                                                            return "<script>alert('用户不存在！');history.go(-1);</script>";
                                                                          } else {
                                                                            // 密码正确
                                                                            if($password == $user['password']) {
                                                                              // 保存到session中
                                                                              Session::set('username', $username);
                                                                              $name = Session::get('username');
                                                                              return "<h1>$name, 您好！登录成功！</h1>";
                                                                            } else {
                                                                              return "<script>alert('密码不正确！');history.go(-1);</script>";
                                                                            }
                                                                          }
                                                                        }
                                                                      }
